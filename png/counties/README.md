Usage
-----

To generate these images with `maps.R` (a script that takes about five minutes):
    1. Move `maps.R` to a directory with a `png/` directory
    2. Set the R directory to that directory: `setwd('path/to/dir')`
    3. Install `ggplot2`, `maps`, `grid` if unable to load. Ex: `install.packages("ggplot2")`
    4. Run ze script `source("maps.R")`
